# Membangun Portofolio Audit di GitHub

## Pendahuluan

Setelah berhasil membuat laporan audit PDF yang profesional, langkah selanjutnya adalah **menunjukkannya ke dunia**. Portofolio di GitHub adalah cara terbaik untuk mendokumentasikan kemampuan Anda sebagai security researcher dan menarik perhatian calon klien atau pemberi kerja.

---

## Langkah-Langkah Membangun Portofolio

### 1. Buat Repositori Baru di GitHub

- Buka GitHub dan klik **New repository**.
- Beri nama repositori (contoh: `updraft-security-portfolio`).
- Pilih visibilitas (public agar dapat dilihat umum).
- Klik **Create repository**.

### 2. Unggah File Laporan

- Pada halaman repositori baru, pilih opsi **upload an existing file**.
- Seret dan lepas file PDF laporan audit ke area yang tersedia.

### 3. Beri Nama File dengan Tanggal yang Jelas

- **Penting:** Sertakan tanggal audit pada nama file.
- Contoh: `2023-12-19 PasswordStore Audit Report.pdf`
- Ini membantu menunjukkan riwayat dan perkembangan Anda.

### 4. Lokasi File PDF

- Jika perlu mencari file PDF di komputer:
  - Di VS Code, klik kanan pada file PDF → **Reveal in File Explorer** (Windows) atau **Reveal in Finder** (Mac).

### 5. Commit Perubahan

- Setelah file terunggah, scroll ke bawah.
- Isi pesan commit (misal: "Add PasswordStore audit report").
- Klik **Commit changes**.

### 6. (Opsional) Tambahkan README

- Buat file `README.md` di repositori.
- Jelaskan isi repositori dan perjalanan keamanan Anda.
- Contoh: Daftar audit yang telah dilakukan, keterampilan yang dimiliki, tautan ke laporan.

---

## Manfaat Portofolio

| Manfaat | Penjelasan |
|---------|------------|
| **Kredibilitas** | Menunjukkan bukti nyata pekerjaan audit. |
| **Aksesibilitas** | Klien atau rekruter dapat melihat hasil kerja Anda kapan saja. |
| **Perkembangan karir** | Portofolio yang terus diperbarui membantu dalam melamar posisi security researcher. |
| **Berbagi pengetahuan** | Orang lain dapat belajar dari laporan Anda. |

---

## Kesimpulan

- **Portofolio GitHub** adalah alat gratis dan efektif untuk memamerkan hasil audit.
- **Beri nama file dengan tanggal** agar mudah dilacak.
- **Tambahkan README** untuk memberikan konteks tentang diri dan pekerjaan Anda.
- Setelah diunggah, laporan dapat dibagikan tautannya ke siapa pun.
